Super Admin

Download all data

Export a complete backup of everything stored in Design Tracker — all projects, features, requirements, statuses, Atlas components, and settings — as a single JSON file.

Use this to back up your data before making large changes, or to transfer it to another device.

Upload all data

Restore a previously downloaded backup. This will replace everything currently in Design Tracker with the contents of the file.

Important: This will overwrite all current data

Uploading a backup replaces all projects, features, requirements, statuses, and settings in this browser. A timestamped backup of the current data will be downloaded automatically before the restore is applied.

Wipe all data

Important: This cannot be undone

Wiping all data will permanently delete every project, feature, requirement, status, Atlas component, and setting stored in this browser. The app will reset to its default demo state.

There is no undo. Download a backup first if you want to keep your data.
